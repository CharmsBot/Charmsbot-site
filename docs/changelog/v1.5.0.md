### `/element`
Quick element efficiency checker for your hunt.

**Features:**
- Paste Hunt Analyzer data
- Shows overall element effectiveness for the spawn
- Per-monster weaknesses and resistances
- Which elements deal bonus damage
- Which elements are resisted

**No character required** - great for quick spawn analysis before hunting.

### `/splitloot`
Split party hunt loot equitably among members.

**Features:**
- Paste your Tibia session data
- Automatic balance calculation per member
- Damage and healing percentage breakdown
- Transfer instructions for settling debts
- Optional extra expenses per member

**Channel restriction:** Must be used in channels named:
- `hunt-analyser`
- `split-loot`
