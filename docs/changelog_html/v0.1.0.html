🚀 First Release

CharmsBot is a Discord bot designed to help Tibia players optimize their charm assignments for hunting.

📊 Available Commands

/analysis

Analyze your hunt using your selected charms.

Features:

  • Paste Hunt Analyzer data from Tibia
  • Optimal charm assignment calculation
  • Uses a fixed 10% proc rate
  • Element efficiency analysis

/character add

Add a new Tibia character to use with the bot.

  • Character name, level, max HP, max mana
  • Select unlocked charms

/character view

View your saved characters and their charm configurations.

/character edit

Edit an existing character's stats or charm selection.

/character delete

Remove a character and all its saved data.

📌 Notes

  • All charm calculations use a fixed 10% proc rate
  • No charm points budget system
  • No charm level selection
