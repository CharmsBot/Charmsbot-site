👤 Character Management

Command Description
/character add Add a new character with stats, charm points, and charms
📸 /character add — Flow
Create a character, add charms by level, and save.
/character view View all your saved characters and their configurations
📸 /character view — Flow
Select a character to view its saved information.
/character edit Edit an existing character's stats and charms
📸 /character edit — Flow
Select a character, update stats, optionally update charms, and save.
/character delete Delete a character and all associated data
📸 /character delete — Flow
Select a character, confirm deletion, and see the result.

📊 Hunt Analysis

Command Description
/analysis Analyze your hunt using your character's saved charms
📸 /analysis — Flow
Select a character, paste your Hunt Analyzer, view results, and save.
/bestcharms Find the optimal charm configuration using ALL available charms
📸 /bestcharms — Flow
Select a character, paste your Hunt Analyzer, view the best setup, and save.
/element Quick element efficiency checker (no character required)
📸 /element — Flow
Paste your Hunt Analyzer and check element weaknesses for the spawn.

📋 Hunt Management

Command Description
/hunt view View your saved analyses and bestcharms configurations
📸 /hunt view — Flow
Select a character, choose the hunt type, then open a saved hunt.
/hunt delete Delete a specific saved hunt
📸 /hunt delete — Flow
Select a character, choose the hunt type, confirm deletion, and see the result.
/hunt rename Rename a saved hunt

🧰 Utilities

Command Description

/weaponxp Calculate weapon proficiency XP from your hunt
/hotcuisine Calculate ingredients for Hot Cuisine Quest
