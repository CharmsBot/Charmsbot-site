### `/analysis`
Analyze your hunt using your selected charms.

**How it works:**
- Paste your Hunt Analyzer data
- Select which charms you want to use for this analysis
- Bot calculates optimal charm-to-creature assignments
- All charms use a fixed **10% proc rate**
- Shows total damage and damage per proc

### `/bestcharms`
Find the best charm configuration using all available damage charms in the game.

**How it works:**
- Paste your Hunt Analyzer data
- Bot evaluates all 9 damage charms automatically
- All charms use a fixed **10% proc rate**
- Shows optimal assignments for maximum damage
